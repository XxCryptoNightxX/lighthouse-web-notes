Selector:
h1		All <h1> elements.

h1, p		All <h1> elements and all <p> elements.

div p		All <p> elements that are inside a <div> element.

div > p		All <p> elements where the parent is a <div> element.

ul + p		The <p> element that are next to a <ul> elements.

ul ~ table	All <table> elements that are siblings of a <ul> element.

::selection	Matches the portion of an element that is selected by a user.

p::first-letter	The first letter of all <p> elements.
